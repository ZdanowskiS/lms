Dodatek Api 2019.06.02

Pozwala na uzycie interfejsu Python/qt zamiast www do wykonywania niektórych czynności

Funkcje:
-Lista zgłoszeń przypisanych do zalogowanego usera
-Lista zdarzeń
-Nowa notatka do zgłoszenia
-Edycja notatki i możliwość rozwiązania zdarzenia
-Podstawowe informacje o kliencie 
-Możliwość przeładowania hosta

Konfiguracja:
-adres www do modułu api LMS wraz z danymi do logowania
-początkowe wymiary okna
