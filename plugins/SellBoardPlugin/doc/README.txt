Dodatek Sell board (giełda) do LMS git 20.02.2019r

Konfiguracja:

sellboard.blockip - lista blokowanych ip rozdzielonych przecinkami
sellboard.allowip - lista dopuszczonych ip rozdzielonych przecinkami

Opis:
*Lista sprzedawców 
Lista osób odpowiadających za sprzedaż danego przedmiotu, zawierajaca podstawowe dane kontaktowe dla zaintersowanych
*Nowy sprzedawca

*Lista kategorii
Oferowane przedmioty można podzielić na kategorię
*Nowa kategoria

*Lista przedmiotów
Lokalna lista dodanych przedmiotów pozwala skontrolować co mamy dodane
*Nowy przedmiot
Przedmioty opisujemy jedynie w podstawowy sposób, następnie dodając im kategorię. Nie jest to sklep, jedynie podstawowa informacja dla osób wiedzacych czego szukają.

*Lista hostów
Lista url pod którymi spodziewamy się znaleźć możliwość odczytania ofert innych firm.
-Sprawdzenie hostów pozwala ustalić które w danej chwili odpowiadają, tylko one będą urzywane
-Pobranie hostów umożliwia pobranie list udostępnianych hostów od innych połaczonych użytkowników
*Nowy host

*Lista ofert
Łączy się ze wszystkimi dostępnymi hostami pobierając kategorie i oferowane przedmioty. Dane będą wyświetlane stopniowo wraz z ich wczytywaniem z kolejnych hostów.
